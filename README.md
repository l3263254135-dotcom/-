# 🍽️ 每日餐食助手 (Daily Meal Assistant)

一个帮助用户解决每天吃饭问题的智能应用，提供餐食建议、营养跟踪和食谱管理。

## 功能特性

- 📋 **餐食规划** - 为每天的早中晚餐提供建议
- 🎲 **随机推荐** - 根据偏好随机推荐餐食
- 📊 **营养分析** - 跟踪卡路里和营养成分
- 🛒 **购物清单** - 自动生成所需食材清单
- ⭐ **收藏食谱** - 保存和管理喜欢的食谱
- 🔍 **搜索功能** - 快速搜索特定菜肴
- 🌍 **多种菜系** - 支持中餐、西餐、日料等多种菜系

## 技术栈

- **前端**：React, TypeScript, Tailwind CSS
- **后端**：Node.js, Express
- **数据库**：MongoDB
- **工具**：Vite, Jest, ESLint

## 快速开始

### 前置要求

- Node.js 16+
- npm 或 yarn

### 安装

```bash
git clone https://github.com/l3263254135-dotcom/-.git
cd daily-meal-assistant
npm install
```

### 开发

```bash
npm run dev
```

### 构建

```bash
npm run build
```

### 测试

```bash
npm test
```

## 使用示例

1. 打开应用首页
2. 选择今天的餐食偏好
3. 获取每餐建议
4. 查看营养信息
5. 添加食材到购物清单
